# Judge Checklist

Use this checklist in the README and video to make scoring obvious.

## Cobo Track

- [ ] Project is centered on Agentic Commerce, not just wallet display.
- [ ] Agent has real or demonstrable funds execution ability.
- [ ] Funding operation goes through CAW.
- [ ] Payment/release operation goes through CAW.
- [ ] Policy boundary is shown.
- [ ] Denied transaction is shown.
- [ ] CAW audit logs are shown.
- [ ] README names CAW key code paths.
- [ ] Demo includes wallet address / tx hash / testnet evidence.

## Z.AI Long-Horizon Add-on

- [ ] GLM-5.1 is used in Worker/Evaluator core flow or adapter is prepared.
- [ ] Trace shows task decomposition.
- [ ] Trace shows tool calls or simulated tool calls.
- [ ] Trace shows validation and fix loop.
- [ ] Final output is more than one-shot text: report + patch + hash + chain action.

## Demo Quality

- [ ] Demo works without relying on a live market condition.
- [ ] No private keys in screen recording.
- [ ] No real funds.
- [ ] Story is understandable in 60 seconds.
- [ ] Failure mode is visible and meaningful.
